# AWS Infrastructure as Code (IaC) Demo

This project showcases a secure, scalable AWS infrastructure provisioned using **Terraform**. It reflects a production-grade environment for a Java microservices application.

## Architecture Highlights
- **VPC Design**: Public and Private subnets across multiple Availability Zones.
- **EC2 Instances**: Auto-scaling groups for high availability.
- **RDS**: Managed PostgreSQL database with automated backups.
- **S3**: Secure object storage for artifacts and logs with lifecycle policies.
- **IAM Policies**: Least-privilege access controls.

## Tools Used
- **Terraform**: Infrastructure provisioning.
- **AWS CLI**: Resource management.
- **CloudWatch**: Monitoring and logging.
